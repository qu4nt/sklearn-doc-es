sklearn.utils.check_random_state¶

sklearn.utils.check_random_state()¶

Convierte semillas en una instancia de np.random.RandomState

Parámetros
seedNone, int o instancia de RandomState

Si la semilla es None, devuelve el singleton RandomState utilizado por np.random. Si seed es un int, devuelve una nueva instancia de RandomState sembrada con seed. Si la semilla ya es una instancia de RandomState, la devuelve. En caso contrario, se produce un ValueError.
