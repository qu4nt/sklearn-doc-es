sklearn.metrics.mean_poisson_deviance¶

sklearn.metrics.mean_poisson_deviance()¶

Pérdida de regresión Desviación media de Poisson (mean Poisson deviance regression loss).

La desviación de Poisson es equivalente a la desviación de Tweedie con el parámetro de potencia power=1.

Leer más en el Manual de Usuario.

Parámetros
y_truearray-like de forma (n_samples,)

Valores objetivo reales (correctos). Requiere y_true >= 0.

y_predarray-like de forma (n_samples,)

Valores objetivo estimados. Requiere y_pred > 0.

sample_weightarray-like de forma (n_samples,), default=None

Ponderaciones de muestras.

Devuelve
lossfloat

Un valor de punto flotante no negativo (el mejor valor es 0.0).

Ejemplos

>>> from sklearn.metrics import mean_poisson_deviance
>>> y_true = [2, 0, 1, 4]
>>> y_pred = [0.5, 0.5, 2., 2.]
>>> mean_poisson_deviance(y_true, y_pred)
1.4260...
