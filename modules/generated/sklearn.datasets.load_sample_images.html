sklearn.datasets.load_sample_images¶

sklearn.datasets.load_sample_images()¶

Carga imágenes de muestra para la manipulación de imágenes.

Carga ambas, china y flower.

Leer más en el Manual de Usuario.

Devuelve
dataBunch

Objeto dictionary-like, con los siguientes atributos.

imageslist de ndarray de forma (427, 640, 3)

Las dos imágenes de muestra.

filenameslist

Los nombres de archivo de las imágenes.

DESCRstr

La descripción completa del conjunto de datos.

Ejemplos

Para cargar los datos y visualizar las imágenes:

>>> from sklearn.datasets import load_sample_images
>>> dataset = load_sample_images()     
>>> len(dataset.images)                
2
>>> first_img_data = dataset.images[0] 
>>> first_img_data.shape               
(427, 640, 3)
>>> first_img_data.dtype               
dtype('uint8')
