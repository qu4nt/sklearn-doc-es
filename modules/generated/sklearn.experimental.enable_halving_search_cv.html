sklearn.experimental.enable_halving_search_cv¶

Activa los estimadores de búsqueda sucesiva a la mitad

La API y los resultados de estos estimadores pueden cambiar sin que se produzca un ciclo de obsolescencia.

La importación de este archivo establece dinámicamente HalvingRandomSearchCV y HalvingGridSearchCV como atributos del módulo model_selection:

>>> # explicitly require this experimental feature
>>> from sklearn.experimental import enable_halving_search_cv # noqa
>>> # now you can import normally from model_selection
>>> from sklearn.model_selection import HalvingRandomSearchCV
>>> from sklearn.model_selection import HalvingGridSearchCV

El comentario # noqa puede eliminarse: sólo indica a los linters como flake8 que ignoren la importación, que aparece como no utilizada.
