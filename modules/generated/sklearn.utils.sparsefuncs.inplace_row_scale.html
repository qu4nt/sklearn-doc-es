sklearn.utils.sparsefuncs.inplace_row_scale¶

sklearn.utils.sparsefuncs.inplace_row_scale()¶

Escalamiento de filas in situ de una matriz CSR o CSC.

Escala cada fila de la matriz de datos multiplicando con la escala específica proporcionada por el invocador asumiendo una forma (n_samples, n_features).

Parámetros
Xmatriz dispersa de forma (n_samples, n_features)

Matriz a escalar. Debe tener formato CSR o CSC.

scalendarray de forma (n_features,), dtype={np.float32, np.float64}

Arreglo de valores precalculados por cada muestra para usar en el escalamiento.
